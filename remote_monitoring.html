24 Hour Remote Alarm Monitoring
 
Rontech Security provides ASIAL approved Grade A1 Monitoring to Australian standards AS2201-2 for the following monitoring communication formats. Our website contains a specific page detailing our Grade A1 Monitoring Station.

  • Digital Communicators (Diallers, GSM)
  • Multi Path (IP, GPRS, PSTN)
  • Securitel ® (Telstra Product)
  • Video Verification

Digital Communicator

This system incorporates an automatic Digital Communicating device that communicates changes of status of your Intruder Alarm system to our Central Monitoring Station via your existing standard telephone exchange line. Be aware that there are a number of security risks involved with Digital Communicator and GSM formats.

The GSM interface unit constantly monitors your standard telephone exchange line and switches to the GSM cellular network in the even of a line failure being detected. Changes of status of your intruder alarm system will then be transmitted to our central monitoring station via the GSM cellular network until your standard telephone exchange line is reinstated.

This type of communication format is recommended for "LOW RISK" security applications.

The alternatives are Securitel® or Multi Path (IP, GPRS, PSTN).

Multi Path (IP, GPRS, PSTN)

This format utilises a number of communication paths to transmit changes in status of your Intruder Alarm system to our Central Monitoring Station such as:

  • IP (ADSL) communication via a connection to the public IP or VPN (Virtual Private Network) service.
  • GPRS communication utilises a polled Radio System which transmits via Rontech Security's VPN network.
  • PSTN communication incorporates an automatic Digital Communicating device via your existing standard telephone exchange line.
This type of communication format is recommended for "HIGH RISK" security applications.

Securitel®

This format incorporates a Telstra Securitel® Interface, which transmits changes of alarm status to our Central Monitoring Station via Telstra Australia's Securitel® Network.

Securitel® is not available in all areas; please check with our sales representative for availability of this service in your area.

This type of communication format is recommended for "HIGH RISK"security applications.

Video Verification

With the combination of an Intruder Alarm and CCTV surveillance system, alarm verification can be achieved by transmitting a video file directly to our remote central monitoring station. This file can then be viewed and actioned accordingly by our monitoring operators.
